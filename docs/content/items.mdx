<p>

Accessories are unique equipment for werewolf player which can be equipped in the Vampirism Menu. They each have a special effect (see [Refinements](https://github.com/TeamLapen/Werewolves/wiki/Refinements) for more information about these effects)

<details><summary>how to acquire</summary>

All accessories receive random refinements.


##### Entity Loot
- Beast Werewolf
- Survivalist Werewolf

##### Chest Loot
- Abandoned Mineshaft
- Desert Pyramid
- Jungle Temple
- Stronghold Corridor
- Stronghold Library

</details>
</p>
